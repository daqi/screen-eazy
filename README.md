# CamShadow MVP

基于 Tauri 的 ScreenSaga 风格 MVP，目标是提供高美感摄像头悬浮体验（纯摄像头模式）。

## 功能

- 透明置顶摄像头浮层（圆角、阴影、边框可调）
- 摄像头浮层可拖拽移动
- 控制中心：摄像头样式调节、浮层显示/隐藏
- 托盘图标：快速呼出控制中心与摄像头浮层

## 启动

```bash
npm install
npm start
```

## 打包后权限（macOS）

- 首次运行安装包版本时，请在“系统设置 → 隐私与安全性”中允许：
	- 摄像头（Camera）
- 若之前拒绝过权限，需在系统设置中手动重新开启后重启应用。
